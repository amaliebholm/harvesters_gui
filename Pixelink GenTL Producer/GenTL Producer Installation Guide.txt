INSTALLING GENTL PRODUCER:

1) Environment path variables to the 32-bit and 64-bit GenTL producers must be created if they do not already exist

2) To create this path go to Control Pannel->System->Advanced System Settings->Advanced and click on "Environment Variables..."

3) If they do not exist, create two new system variables called GENICAM_GENTL32_PATH and GENICAM_GENTL64_PATH

4) Set value of the paths to the folders containing the 32-bit and 64-bit Pixelink GenTL producers ( TLpixelink32.cti and TLpixelink64.cti )

5) Copy "SphinxLib.dll" found in "SphinxLib 64-bit" to C:\Windows\System32

6) Copy "SphinxLib.dll" found in "SphinxLib 32-bit" to C:\Windows\SysWOW64




INSTALLING HARVESTER (and Python):

1) Harvester only works in 64-bit Python installations up to Python version 3.7

2) An installer for Python 3.7 64-bit is available here -> https://www.python.org/downloads/release/python-370/

3) When installing Python 3.7 ensure that "Pip" is selected to be installed as well

4) To install Harvester open a Command Promt and enter "pip install -U harvesters"

4a) Harvester sample app also requires running "pip install pillow", "pip install scikit-image", and "pip install numpy"

5) To make sure that harvester installed correctly open a Python command windows and enter "from harvesters.core import Harvester". This line should execute without issue or warning
